macOS下搭建Flutter开发环境

2020-09-07 3 min read # Flutter # 学习路线

首先要下载必要的文件

我选择开发工具的是Android Studio，因为我本身就是开发Android的方便使用我就直接使用Android Studio了，当然你们还可以选择IDEA或者Visual Studio

  • Flutter sdk
  • Android Studio

安装flutter sdk

解压上面下载的sdk到你的工作目录，我这里就放到了Users/xxx目录下

然后把环境变量添加到base_profile里面，我们使用vi工具操作（我一般习惯性用vi，你们熟悉其他编辑器也可以使用的）

vi ~/.bash_profile


添加刚刚解压的flutter sdk的目录到path

export PATH=$PATH:/Users/xxxx/flutter/bin

这个是我当前解压的目录，xxxx为用户名，仅作参考，其他目录也是一样的

然后还需要加入两个镜像地址，由于在国内访问Flutter可能会受到限制，谷歌官方为中国开发者搭建的临时镜像（临时镜像不保证一直可以使用）

export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn

保存退出然后运行

source ~/.bash_profile

这样环境变量就弄好了，如果重启失效可以参考 macOS配置adb 最后特别提醒

终端运行flutter doctor检查flutter环境安装情况

flutter doctor


运行后如图有flutter版本号就是安装成功了，下方有列举了一些开发工具安装情况，打x的表示没有安装，打勾表示安装好了的，例如我这里Android Studio已经打勾了就安装好flutter插件了。

开发工具安装flutter插件

我这里举例Android Studio插件安装

我们需要两个插件

  1. flutter插件：提供程序的运行、调试和热重载等功能
  2. Dart：提供Dart语言的支持

首先打开Android Studio，找到preferences --> plugins --> 搜索flutter，如图

我这里已经安装好了，没有安装的直接点击install即可，一会会弹出同时安装Dart，点击yes就可以了
