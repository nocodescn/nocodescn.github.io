macOS配置adb

2020-09-05 1 min read # macOS

  • 1.首先打开终端命令行，我们检查一下是否有配置好adb命令，直接终端输入adb提示command not found: adb就是没有配置

  • 2.下面我们在用户目录下输入

vi ~/.bash_profile
  • 3.然后用vi工具直接键入导入adb命令的目录
export PATH=${PATH}:/Users/目录/Library/Android/sdk/platform-tools

此处『目录』为你电脑的用户名，如下图第一行所示，保存退出vi工具

如果你电脑的adb安装在其他目录，根据你安装的目录进行修改即可

  • 4.用source命令执行一下刚刚新建的文件
source ~/.bash_profile
  • 5.好了，到这里就大功告成了，执行一下adb看看是否配置成功

特殊情况

如果你的电脑重启后adb命令不生效了，可以修改.zshrc文件，重启自动source ~/.bash_profile

vi打开.zshrc文件，此时如果电脑没有这个文件会自动创建

vi ~/.zshrc


最后一行输入

source ~/.bash_profile


保存后退出即可。
