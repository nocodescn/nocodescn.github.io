AndroidStudio Git提交错误：fatal- could not read Username for 'xxx'- Device not configured

2022-07-10 1 min read

AndroidStudio的git一直用着好好的，更换了自己部署的git仓库后就出问题了

表现为

  • Android Studio更改git remote时会提示该错误。
  • 使用终端修改remote后，提交代码也会提示该错误

Google了一下找到的答案：
https://stackoverflow.com/questions/40274484/fatal-could-not-read-username-for-https-github-com-device-not-configured

这里我使用的是MacOS，Windows应该也差不多的

确保Credentials Helper勾选上即可解决。
